## Architectural Decisions (read before writing any code)

**AFS identifiers:** Only AFS-0001 exists as a published specification. Do not invent AFS-0002 through AFS-0010. `implementedSpecifications` contains only real published entries. A pending notice communicates that the table will grow as specifications are published.

**Repository structure:** The `repository` section describes architectural domains, not filesystem paths. Filesystem paths change; domains are architectural. The section is intentionally stable — it maps to the layer model, not to a directory listing.

**Architecture mapping fields:** `canonicalPackage` (not `referencePackage`) holds the scoped npm package identifier (`@rohinik-org/runtime` etc.). Column header: "Canonical Package". `authority` is a structured object `{ label, specification }` — column header: "Governing Authority", display as `"{label} — {specification}"`. All rows currently use `{ label: 'Foundation Constitution', specification: 'AFS-0001' }` because that is the only published specification.

**Coverage status values:** `implemented | partial | planned` — not `full`. "Implemented" is objective; "full" is subjective and implies a completeness percentage.

**Lifecycle connectors:** No `→` text arrows. The pipeline uses CSS `border-r` on each step cell as the visual connector, consistent with the Governance page visual grammar.

## Codebase Conventions (read before writing any code)

- **Sections**: always `<section id="..." aria-labelledby="...-heading">` wrapping `<div className="max-w-site mx-auto px-[var(--spacing-margin-edge)] py-16 md:py-24">`
- **H2**: `font-headline text-headline-md font-bold text-on-surface`
- **H3**: `font-headline text-headline-sm font-semibold text-on-surface`
- **Body text**: `font-body text-body-md text-on-surface-variant leading-relaxed`
- **Mono labels**: `font-mono text-label-caps uppercase tracking-widest text-on-surface`
- **Muted mono**: `font-mono text-label-caps uppercase tracking-widest text-on-surface-variant`
- **2-col grid row**: `grid grid-cols-1 md:grid-cols-[Xrem_1fr] gap-2 md:gap-8 py-5 md:items-baseline`
- **Divider list**: `divide-y divide-outline-variant border-t border-outline-variant`
- **Border section bg**: `bg-surface-container-low` (alternating sections)
- **No rounded corners, no shadows, no hex colours, no `'use client'`, no text arrow characters as structural connectors**
- **Tests**: sync render (NOT async), no `!` non-null assertions, use `?.` or guarded `if` blocks
- **Page function**: sync `function RSOnePage()` — NOT async (jsdom can't render async RSC)
- **Content import path**: `@/content/reference-standards/rs1` (note: subdirectory `reference-standards/`)
- **Component import path**: `@/components/reference-standard/ComponentName`
